# Auratten Web

A professional, high-performance attendance management dashboard built with Next.js. Auratten provides real-time tracking, institutional analytics, and seamless reporting for modern educational environments.

## Features

- **Real-time Dashboard**: Overview of active sessions, total users, and attendance rates.
- **Reporting Engine**: Generate detailed PDF and CSV reports for institutions and individual students.
- **Role-based Access**: Tailored experiences for Administrators, Teachers, and Students.
- **Modern UI**: Clean, responsive design built with Tailwind CSS and CSS Modules.

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **State Management**: [Zustand](https://docs.pmnd.rs/zustand/)
- **Styling**: Tailwind CSS & Vanilla CSS
- **Icons**: Lucide React
- **Charts**: Recharts
- **PDF Generation**: jsPDF & autoTable

## Getting Started

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Environment Setup**:
    Create a `.env.local` file with the following:
    ```env
    NEXT_PUBLIC_API_URL=your_api_url
    ```

3.  **Run the development server**:
    ```bash
    npm run dev
    ```

4.  **Build for production**:
    ```bash
    npm run build
    ```
